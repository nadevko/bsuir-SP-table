# Лабораторная №3

## Задание

1. Сделать ячейки таблицы кликабельными с помощью мыши.
2. Реализовать возможность изменения содержимого ячеек по двойному клику.
3. Рассчитать суммарный размер:
   - первое значение — в байтах;
   - второе значение — перевести в мегабайты (или другую ближайшую единицу);
   - третье значение — реальный размер на диске (в блоках).
4. Определить суммарный размер файлов по блокам памяти.
5. Посчитать фактический размер файлов (сумму содержимого).
6. При нажатии изменять значения ячеек (обработка событий).
