# Лабораторная №4

## Задание

1. Синхронизация двух каталогов
2. Берём каталог и подкаталоги рекурсивно
3. Чекаем другой каталог
   1. Если в dir1 есть файл находим соответствующие в dir2
   2. Если файла нету, то копируем его в dir2 из dir1
   3. Если подкаталога нету, создаём
   4. Сравнивать файлы не по именам, выводим файлы идентичны и не копируем
   5. Число файлов должно совпасть
   6. Копировать через буффер open->read->write
4. Слева показываем исходный каталог, справа новый, четыре колонки:
   1. имя dir1
   2. размер dir1 (в байтах)
   3. имя dir2
   4. размер dir2 (в байтах)
