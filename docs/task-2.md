# Лабораторная №2

## Задание

1. В ячейки таблицы нужно будет вывести файлы и их аттрибуты в заданном каталоге
2. При запуске указываем каталог
3. Если файлов много и они не помещаются, нужна прокрутка
   1. Как минимум, постраничная
   2. Круче - мышкой
4. Выводим все файлы рекурсивно - в подкаталогах тоже
5. Формат записи (именно в таком порядке):
   1. Размер в байтах
   2. Дата (число.месяц.год)
   3. Права доступа (+sticky bit) (восьмирично или буквами)
